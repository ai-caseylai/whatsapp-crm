# WhatsApp 媒體展示使用說明

## 📱 功能介紹

WhatsApp CRM 系統現在包含三種不同的媒體展示方式，讓你可以方便地查看所有圖片、視頻和 Emoji 消息。

## 🎯 訪問方式

### 1. 簡易媒體瀏覽器（推薦）
**訪問網址**: https://whatsapp-crm.techforliving.app/media-simple.html

**特點**:
- ✅ 無需登錄
- ✅ 快速載入
- ✅ 展示最近的圖片、視頻和 Emoji 消息
- ✅ 優雅的卡片式設計
- ✅ 自動標記你發送的消息

**適用場景**: 快速瀏覽最近的媒體內容

---

### 2. 完整媒體畫廊
**訪問網址**: https://whatsapp-crm.techforliving.app/media.html

**特點**:
- 🔒 需要認證（自動使用密碼）
- 📊 顯示統計信息（總數、圖片數、視頻數、Emoji數）
- 🎨 支持篩選（全部、圖片、視頻、Emoji）
- 🖼️ 從數據庫動態載入媒體
- ⚡ 實時更新

**適用場景**: 需要詳細查看和篩選媒體內容

---

### 3. 高級媒體畫廊（開發中）
**訪問網址**: https://whatsapp-crm.techforliving.app/media-gallery.html

**特點**:
- 🎬 支持 Lightbox 放大查看
- 🎥 視頻預覽和播放
- 🔍 高級篩選和搜索
- 📱 完全響應式設計

**狀態**: API 接口開發中

---

## 📸 支持的媒體類型

1. **圖片** 🖼️
   - JPG, PNG, WebP 格式
   - 自動縮略圖
   - 點擊放大查看

2. **視頻** 🎥
   - MP4, WebM 格式
   - 內聯播放控制
   - 預覽縮略圖

3. **Emoji** 😊
   - 所有 Unicode Emoji
   - 大號顯示
   - 反應消息（❤️、👍、🙏 等）

4. **文本消息** 💬
   - 包含 Emoji 的文本
   - 引用和回復消息
   - 格式化文本

---

## 🎨 界面特點

### 視覺設計
- 🌈 漸變背景（紫色到藍色）
- 💳 白色卡片設計
- ⚡ 懸停動畫效果
- 📱 響應式布局

### 信息展示
每個媒體項目顯示：
- 發送者姓名
- 對話/群組名稱
- 發送時間（精確到分鐘）
- 消息類型標識
- "你發送的" 標記（自己的消息）

## 🔧 技術說明

### 數據來源
- 主數據庫：Supabase
- 實時同步：Baileys WhatsApp Web API
- 媒體存儲：服務器本地文件系統

### API 端點
- `/api/admin/media/list` - 獲取媒體列表
- 支持參數：
  - `limit`: 限制數量（默認 50）
  - `type`: 媒體類型（image/video/emoji）
