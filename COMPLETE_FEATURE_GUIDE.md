## 📐 图片编辑

### 快捷编辑按钮

当选择图片后，会显示以下快捷按钮：

#### 分析功能（白色按钮）
- **🔍 分析图片**：全面分析图片内容
- **📄 文字提取**：OCR 识别图片中的文字

#### 编辑功能（渐变按钮）
- **🎨 移除背景**：自动识别主体并去除背景
- **☀️ 提高亮度**：让图片更明亮
- **🔆 增强对比度**：使图片更清晰
- **🌫️ 模糊效果**：添加模糊特效
- **✨ 锐化图片**：增强图片细节

### 使用流程

```
1. 在 WhatsApp 聊天中选择图片
   ↓
2. 点击「图片分析」按钮
   ↓
3. 图片显示在 Gemini 侧边栏
   ↓
4. 点击快捷编辑按钮或输入自定义命令
   ↓
5. 等待 AI 处理（10-40秒）
   ↓
6. 查看编辑后的图片
   ↓
7. 发送到 WhatsApp / 下载 / 查看大图
```

### 自定义编辑命令

除了快捷按钮，您还可以输入自定义编辑指令：

**简单编辑：**
```
移除背景
提高亮度
增强对比度
裁剪成正方形
```

**组合编辑：**
```
移除背景并添加柔和阴影
提高 30% 亮度并增强对比度
裁剪成正方形并添加白色边框
```

**风格转换：**
```
转换成黑白风格
添加复古滤镜效果
转换为水彩画风格
转换为油画风格
```

**创意编辑：**
```
把这张照片转换成动漫风格
添加电影级调色效果
转换成极简主义风格
```

## 🎨 图片生成

### 打开图片生成器

点击 Gemini 侧边栏输入区下方的 **「生成图片」** 按钮

### 生成器界面

#### 1. 图片描述（必填）
详细描述您想要生成的图片。描述越详细，效果越好。

**示例：**
```
一只可爱的柴犬在公园里玩耍，阳光明媚，摄影风格
未来科技城市夜景，霓虹灯光，赛博朋克风格
美味的食物摆盘，餐厅级别，专业食物摄影
寧靜的山水風景，中國水墨畫風格，意境深遠
```

#### 2. 寬高比（可選）
- **1:1** - 正方形（适合社交媒体头像）
- **16:9** - 横向（适合横幅、海报）
- **9:16** - 直向（适合手机屏幕、Stories）
- **4:3** - 标准（传统照片比例）
- **3:4** - 竖版（竖版海报）

#### 3. 风格（可选）
- **自动** - 让 AI 自动选择最适合的风格
- **写实照片** - 逼真的摄影效果
- **插画** - 手绘插画风格
- **水彩画** - 艺术水彩效果
- **油画** - 经典油画风格
- **动漫** - 日本动漫风格
- **极简风格** - 简洁现代
- **复古** - 怀旧复古风

### 快速模板

点击预定义模板快速填充描述：

- 🐕 可爱宠物
- 🌃 科技城市
- 🍽️ 美食摄影
- 🏔️ 山水画
- 💼 商务场景
- 🎨 抽象艺术

### 生成流程

```
1. 打开图片生成器
   ↓
2. 输入详细的图片描述
   ↓
3. 选择寬高比和风格（可选）
   ↓
4. 点击「🎨 生成图片」
   ↓
5. 等待生成（约 30-60 秒）
   ↓
6. 查看生成的图片
   ↓
7. 发送到 WhatsApp / 下载 / 继续编辑
```

### 生成提示技巧

#### ✅ 好的描述
```
✓ "一只金毛犬坐在海边看日落，温暖的橙色光线，专业摄影，景深效果"
✓ "现代极简风格的客厅，白色和木色搭配，大落地窗，自然光，室内设计杂志风格"
✓ "赛博朋克风格的街道，霓虹灯招牌，雨后湿润的地面，夜景，电影镜头感"
```

#### ❌ 不够好的描述
```
✗ "一只狗" （太简单）
✗ "漂亮的房间" （不具体）
✗ "街道" （缺少细节）
```

#### 描述要素
1. **主体**：要生成什么？
2. **环境**：在哪里？什么背景？
3. **风格**：什么艺术风格或摄影风格？
4. **光线**：什么样的光线效果？
5. **细节**：特殊的细节或氛围？

## 🎓 最佳实践

### 图片编辑最佳实践

#### 1. 分步编辑
对于复杂的编辑需求，建议分多次进行：

```
❌ 不推荐：
"移除背景，提高亮度，增强对比度，添加阴影，裁剪成正方形"

✅ 推荐：
第一步："移除背景"
第二步："提高 20% 亮度并增强对比度"
第三步："添加柔和阴影"
第四步："裁剪成正方形"
```

#### 2. 使用具体参数
```
❌ "调亮一点" → ✅ "提高 30% 亮度"
❌ "模糊一下" → ✅ "添加轻微高斯模糊"
❌ "改改颜色" → ✅ "增加 15% 饱和度"
```

#### 3. 先测试再批量
在批量处理之前，先对一张图片测试效果：
1. 选择一张代表性图片
2. 尝试不同的编辑命令
3. 找到最佳效果
4. 记录下来用于批量处理

### 图片生成最佳实践

#### 1. 详细描述
```
基础: "一只猫"
进阶: "一只橘色的猫咪"
高级: "一只可爱的橘色英短猫咪，坐在窗台上，阳光洒在它身上，温暖的氛围，专业宠物摄影"
```

#### 2. 风格一致性
如果要生成系列图片，保持描述模板一致：

```
模板: "[主体]，[环境]，[风格关键词]，[光线]，专业摄影"

应用：
- "蛋糕，白色背景，清新风格，柔和光线，专业摄影"
- "咖啡，白色背景，清新风格，柔和光线，专业摄影"
- "面包，白色背景，清新风格，柔和光线，专业摄影"
```

#### 3. 使用参考风格
提及具体的艺术风格或摄影师风格：
```
- "宫崎骏动画风格"
- "赛博朋克风格"
- "北欧极简设计"
- "中国传统水墨画"
```

### 性能优化

#### 1. 避免高频请求
```
✅ 合理使用：
- 每次编辑间隔 10 秒以上
- 批量处理使用脚本而非手动

❌ 避免：
- 连续快速点击编辑按钮
```

#### 2. 选择合适的分辨率

```
用途建议：
```

## ❓ 常见问题

### Q1: 编辑后只返回文字没有图片？

**原因**：模型理解为分析而非编辑

**解决方法**：
1. 使用快捷编辑按钮（效果最稳定）
2. 在命令中明确说明「生成新图片」
3. 使用编辑模板中的预定义命令

**示例**：
```
❌ "处理这张图"
✅ "移除背景并返回新图片"
✅ 点击快捷按钮「🎨 移除背景」
```

### Q2: 生成的图片不符合预期？

**原因**：描述不够具体或风格不明确

**解决方法**：
1. 增加更多细节描述
2. 明确指定风格
3. 使用快速模板作为起点
4. 多次尝试不同的描述

**改进示例**：
```
❌ "一个房间"
→ "一个现代简约风格的客厅"
→ "一个现代简约风格的客厅，白色墙壁，木地板，大落地窗，自然光充足"
→ ✅ "一个现代简约风格的客厅，白色墙壁，原木色地板，大落地窗，下午阳光洒进来，室内设计杂志摄影风格"
```

### Q3: 处理时间太长？

**正常范围**：
- 分析：3-8 秒
- 简单编辑：10-20 秒
- 复杂编辑：20-40 秒
- 图片生成：30-60 秒

**如果超时**：
1. 检查网络连接
2. 简化编辑指令
3. 稍后再试（可能是 API 繁忙）

### Q4: API 错误码含义？

| 错误码 | 含义 | 解决方法 |
|-------|------|---------|
| 401 | API Key 无效 | 检查 `.env` 文件中的 API Key |
| 402 | 余额不足 | 前往 OpenRouter 充值 |
| 429 | 请求频率过高 | 等待 1-2 分钟后再试 |
| 500 | 服务器错误 | 稍后重试或联系支持 |

### Q5: 如何下载编辑后的图片？

**方法 1**：点击「💾 下载」按钮

**方法 2**：
1. 点击「🔍 查看大图」
2. 右键点击图片
3. 选择「图片另存为...」

### Q6: 可以批量处理图片吗？

**当前版本**：暂不支持批量自动处理

**替代方案**：
1. 确定最佳编辑命令
2. 逐张处理（使用快捷按钮提高效率）
3. 等待未来版本的批量处理功能

### Q7: 编辑后的图片质量会下降吗？

**答案**：通常不会，Nano Banana Pro 会尽可能保持原始分辨率

**建议**：
- 原图质量越好，编辑效果越好
- 避免多次重复编辑同一张图
- 如需高质量，从高分辨率原图开始

### Q8: 生成的图片有水印吗？

**答案**：没有水印，生成的图片可以自由使用

### Q9: 能否保存常用的编辑命令？

**当前方式**：使用编辑模板功能

**未来计划**：
- 自定义模板保存
- 编辑历史记录
- 收藏功能

### Q10: 支持哪些图片格式？

**输入格式**：
- ✅ JPG / JPEG
- ✅ PNG
- ✅ WEBP

**输出格式**：
- PNG（默认）
